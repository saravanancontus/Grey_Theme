Quick start

The quickest way to get started is to unzip the file you downloaded and copy the contents into a folder called 'hdflvplayer' at the root of your website. The end result should be:

  • {webroot}/hdflvplayer/skin
  • {webroot}/hdflvplayer/xml
  • {webroot}/hdflvplayer/email.php
  • {webroot}/hdflvplayer/download.php
  • {webroot}/hdflvplayer/hdplayer.swf
  • {webroot}/hdflvplayer/index.html
  • {webroot}/hdflvplayer/readme.txt
  • {webroot}/hdflvplayer/swfobject.js
  • {webroot}/hdflvplayer/videos
  • {webroot}/hdflvplayer/images

Copy the following code onto your webpage. That's it! You should see a player on your site.

  1. 	<script type="text/javascript" src="swfobject.js"></script>
  2. 	<div id="mediaspace"></div>
  3. 	<script type="text/javascript">
  4. 		var s1 =new SWFObject('hdplayer.swf','player','640','360','9');
  5. 		s1.addParam('allowfullscreen','true');
  6. 		s1.addParam('allowscriptaccess','always');
  7. 		s1.addParam('wmode','transparent');
  8. 		s1.write('mediaspace');
  9. 	</script>
